# Web Application for Movie Rating

Full-stack Django web application for giving user ratings to movies.

![Home Page](Movie-List-Home-Page.png)

## Features
- User registration and authentication
- Movie addition by admin
- Movie listing and detail pages
- Per-user movie ratings
- Data storage using PostgreSQL
- Production deployment on Railway

## Tech Stack
- Backend: Python Django
- Database: PostgreSQL
- Frontend: Django Templates, HTML, CSS
- Deployment: Railway
- Version Control: GitHub

## Live Demo
[https://your-railway-url.railway.app](https://web-production-ac023.up.railway.app/)

## Setup (Local)
1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Set DATABASE_URL in settings.py
5. Run migrations
6. Start the server
